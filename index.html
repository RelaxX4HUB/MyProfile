About

Most of my time is spent reverse engineering software and digging into system internals. I like taking things apart to see how they work, especially when the documentation doesn't exist or is deliberately vague. There's something satisfying about understanding a complex system from the ground up.

I build my tools primarily in Rust and C++, and use C# for game engine development. When I'm bored, I write process manipulation tools and other fun stuff - though it's all private software. My focus is usually on security research, vulnerability analysis, and building tools that help other researchers do their work more effectively.

When I'm not buried in a disassembler, I contribute to open source projects and research new attack techniques. I believe the best defense comes from understanding how attacks actually work, so I spend a lot of time on the offensive side of security.
